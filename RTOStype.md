Hard Real-Time operating system
  - System where deadline is the top priority. Once miss consider as total failure.(DEstruction Occur)
    eg. rocket launching sequence, airbag deploy and dose intake control
 
Soft real-time operating system
  - Priority based-system where the system despite executed way past the deadline, the result can still be used.
    e.g youtube uploading video, banking system and live score update.

Firm Real-time Operating System
 -  System where deadline is important, whereby if the deadline is miss the result would consider failure.
    e.g stock market bidding system and weather forecasting
